static <TextendsEntity> T bubolo.world.Entities.create ( Class< T >  c) throws GameLogicException, IllegalStateException
static

Performs the following actions:

  1. A new Entity of the specified type is created.
  2. The new Entity is added to the World, using World.add(Entity).
  3. A new Sprite is created and added to the Sprites list.
  4. One or more Controllers are created and added to the Controllers list.
Parameters
cthe entity's class object. For example, to create a new Tank, call this method using the following form: Entities.create(Tank.class).
Returns
reference to the new entity. Note that the entity has already been added to the World.
Exceptions
GameLogicExceptionif the Entity cannot be instantiated.
IllegalStateExceptionif Entities has not been initialized.
